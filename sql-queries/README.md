# SQL Deep Dive
#### The SQL stage of this project was where I took the raw data, put it into MySQL (Workbench), fixed missing values, errors, redundant columns, and much more. I then took the cleaned data, and did a collection of analytical queries for exploratory analysis. When I felt satisfied with these queries, I moved the cleaned table to Excel to create tables and charts. 
#### The comments within the SQL file go more into detail about WHY I did what I did and HOW I decided to do it.
